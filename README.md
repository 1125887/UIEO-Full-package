# UIEO-Full-package
The pakages of all HarmonyOs 4.3's Huawei system component and HMS Core
Before proceeding, confirm that your base system package version is 104.2.0.182 or above, and also 104.2.0.121.​
(Dial *#*#2846579#*#*→ Board Information → Version Info to check.)
① Use Shizuku to activate any third-party installer (recommended: AIWanJi Toolbox). Install all .apkand .apksfiles from the repository, and move all .hwtfiles into Huawei/Themes.
② Reboot the device. Open the Themes app and apply the "Boundless" theme. All done.
tips:If you encounter issues after installing certain .apkor .apksfiles — such as a black screen on the system UI, apps failing to open, etc. — you can call out to Celia to open Settings, then uninstall the updates for the problematic software.
